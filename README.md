# Apothecary 

A wizard-themed, AI-powered healthcare platform where dragons seek medical guidance from the Grand Wizard. Built for hackathons with a focus on immersive fantasy aesthetics and practical medical AI.

##  Features

- **Landing Page** - Animated rune circles, dragon sprite, and mystical entrance
- **Authentication** - Stone archway-styled login/signup with glowing inscription fields
- **Dashboard** - Wizard's observatory with sidebar navigation and parchment workspace
- **Text Diagnosis** - AI-powered medical guidance with typewriter animation
- **Fantasy UI** - Complete with sprite animations, magical effects, and immersive design

##  Design System

- **Colors**: Deep indigo backgrounds, ember-orange accents, parchment surfaces
- **Typography**: Cinzel serif for headings, Inter sans-serif for body
- **Animations**: Rune rotation, dragon blink, sparkle effects, icon pulses, typewriter text
- **Theme**: Nexus Games minimalism meets Squarespace polish

##  Tech Stack

- React + TypeScript + Vite
- TailwindCSS for styling
- Alchemy-Backend
- Gemini AI for medical guidance
- Custom animations and sprite graphics

##  Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

##  Hackathon Ready
